# 两栏自适应布局

如上图左右两边等高，其实可以看成是列表一样的布局，核心 css 代码如下

.left,
.right {
  display: table-cell;
  /* 控制内容的对齐方式 top middle bottom */
  vertical-align: top;
}

文字部分需要沾满容器的剩余空间，我们只需要设置其宽度为 width: 99999px

左边使用浮动 float: left，右边使用 display: table-cell 同样能达到上边一样的布局，如下
